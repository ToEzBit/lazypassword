# LazyPassword

A simple and interactive command-line password manager application written in Go.

## Installation

### Option 1: Homebrew (Recommended)

```bash
brew tap toezbit/lazypassword
brew install lazypassword
```

### Option 2: Go Install

```bash
go install github.com/toezbit/lazypassword@latest
```

### Option 3: Download Binary

Download the latest binary from [GitHub Releases](https://github.com/toezbit/lazypassword/releases)

### Option 4: Build from Source

```bash
git clone https://github.com/toezbit/lazypassword.git
cd lazypassword
go mod tidy
go build -o lazypassword
```
